# Archived

Completed bugs and features, moved here from BACKLOG.md to keep the backlog clean.

## Bugs
- **Input lag / slow direction change** — queued direction system buffers up to 3 keypresses per tick so rapid inputs are processed sequentially rather than dropped.

## Features
- **End screen on death** — score summary screen with [R] Play Again / [Q] Quit options instead of instant exit.
- **ASCII goodbye screen** — stylized farewell screen shown when the player quits, auto-dismisses after 2.5 seconds.
- **Startup screen** — Amiga demo-style welcome screen with rainbow color bars, block-letter SNAKE logo, snake decoration, and press-any-key prompt.
- **Amiga ASCII aesthetic** — all screens (game board, game over, goodbye, startup) use neon ANSI colors, rainbow bars, and retro block-letter art.
