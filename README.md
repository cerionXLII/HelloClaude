# HelloClaude

A toy project for learning and experimenting with Claude Code.

## Snake Game

A classic Snake game that runs directly in the Windows CMD terminal — no external packages required.

**Requirements:** Python 3.x on Windows (uses `msvcrt` and `ctypes` from the standard library).

**Run:**
```
python snake.py
```

**Controls:**

| Key | Action |
|-----|--------|
| W / Arrow Up | Move up |
| S / Arrow Down | Move down |
| A / Arrow Left | Move left |
| D / Arrow Right | Move right |
| Ctrl+Z | Quit |

> **Note:** Run in a Windows CMD window for best results. PowerShell may handle ANSI escape codes differently.
